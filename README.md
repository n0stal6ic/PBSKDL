# PBSKDL
Python script to download PBS Kids video streams and parse to N_m3u8DL-RE.

# How to use
1. Download the PBSKDL.py file and place in the same directory as the N_m3u8DL-RE binary.
2. Launch the PBSKDL.py file and provide a URL to the video from pbskids.org
3. Chose the video profile that you want (usually the first one) and then let it download.
